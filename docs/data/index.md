# Raw Data

Full detection research results. These contain per-URL metrics including HTTP status codes, content hashes, timing data, and resource measurements.

## Result files

- [Chrome headful vs headless](results-chrome.md) -- detection signal results across 20+ tests comparing Chrome headful and headless modes
- [Chromium](results-chromium.md) -- equivalent results using Chromium instead of Chrome
- [Headless Shell](results-headless-shell.md) -- full comparison of `chrome-headless-shell` against Chrome headless across 965 URLs
- [Headless Shell tables](results-headless-shell-tables.md) -- tabular summaries of headless-shell divergence data (HTTP status, content hash, protocol errors)
